# clone_bank_app

해당 프로젝트는 국민은행 안드로이드 어플 클론코딩 프로젝트 입니다.

##### Front-end
Flutter 오픈 소스 프레임워크를 활용하여 국민은행 스타뱅킹 어플 기능을 클론코딩 하는 방향으로 진행하였으며,
기능에 따라 필요한 부분들을 구현하였습니다.

##### Back-end
node.js와 MySQL을 사용하여 은행 데이터를 만들고, express서버에 연결하는 기능을 구현하였습니다.
